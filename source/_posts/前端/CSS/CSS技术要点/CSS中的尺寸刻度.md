## 绝对单位
- in：英寸
- cm：厘米
- mm：毫米
- pt：磅，1pt=1/72 in
- pc：pica，1pc=12pt

## 相对单位
- px：像素，与设备分辨率挂钩
- rem：与根元素的字号挂钩
- em：与父元素的字号挂钩
- ex：与元素字体的“x 字母高度”挂钩

## 百分比
- vw/vh：相对于视窗的宽/高度的1/100
- vmin/vmax：vw/vh中的较大者
- %：相对另一值的百分比
